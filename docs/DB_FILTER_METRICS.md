# 카드의요정 질문-필터 지표 정의 (V2)

작성일: 2026-03-03  
범위: 현재 질문(`app/question_bank.py`) + 현재 카드 DB(`data/cards.db`, 361건)

## 1) 목적
- 질문 응답에 따라 남은 카드 수가 단계적으로 감소하는지 관리한다.
- 해외결제/마일리지/생활혜택/간편결제/차량/교통 조건의 필터 기여도를 측정한다.
- 인터넷전문은행 주거래 여부 질문은 필터가 아니라 카드사 가점(정렬) 품질로 측정한다.

## 2) 핵심 지표

### KPI-1. 단계별 잔존 카드 수 (Step Remaining)
- 정의: 각 질문 답변 직후 남은 카드 수
- 목표: 단계가 진행될수록 단조 감소(동일 허용)
- 최종 추천 목표 범위: 5~60건

### KPI-2. 단계별 축소율 (Reduction Rate)
- 정의: `(이전 단계 카드 수 - 현재 단계 카드 수) / 이전 단계 카드 수`
- 목표: 필터 질문 평균 축소율 8% 이상

### KPI-3. 해외결제 혜택 보유 카드 수 (Overseas Benefit Count)
- 정의: 현재 후보군 중 해외결제 관련 혜택 키워드를 가진 카드 수
- 키워드: `외화결제`, `해외`, `해외이용`, `해외결제`, `직구`
- 화면 노출: `현재 후보 중 해외결제 혜택 보유 카드: N개`

### KPI-4. 마일리지 적합 카드 수 (Mileage Fit Count)
- 정의: `mileage_interest=Yes` 선택 시 남는 카드 수
- 추가 분기: `대한항공/그 외` 후속 질문 적용 시 추가 축소율

### KPI-5. 생활혜택 커버리지 (Category Coverage)
- 대상 질문: 베이커리/카페, 식당/배달, 쇼핑, 편의점, 통신, OTT/스트리밍, 간편결제, 차, 대중교통
- 정의: 각 질문 `Yes` 선택 시 실제로 후보군이 감소하는 비율

### KPI-6. 가점 정렬 품질 (Preference Score Impact)
- 정의: 인터넷전문은행 질문에 따른 가점 적용 후 상위 N개 카드의 카드사 분포 변화
- 규칙:
  - Yes: 현대/삼성/롯데/BC +2
  - No: 국민/우리/신한/하나 +2

## 3) 현재 필터/가점 규칙

### 연회비 질문
- 옵션:
  - 3만원 이상, 3만원 이하
  - 10만원 이상, 10만원 이하
  - 기타 직접 입력: 입력값 ±1만원 범위

### 해외여행/직구 (Y/N)
- Yes면 해외결제 관련 키워드가 있는 카드만 유지

### 마일리지
- 관심 Yes면 마일리지 관련 카드만 유지
- 이후 대한항공/그 외 질문으로 추가 필터

### 생활영역 질문들
- Yes일 때 각 영역 키워드가 있는 카드만 유지
- No일 때 해당 질문은 필터 미적용

### 인터넷전문은행 질문
- 필터 미적용
- 카드사 그룹별 가점만 적용해 정렬 순서를 변경
