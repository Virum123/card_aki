# Notion 연동 계획 (구현 보류)

작성일: 2026-03-03
상태: 계획만 정리, 구현은 추후 별도 진행

## 목적
- 설문 세션/추천 결과를 노션 DB에 적재해 운영 지표를 관리한다.

## 연동 방식 후보
1. 결과 로그 적재형 (우선)
- 설문 완료 시 `응답/남은 카드 수/상위 추천 카드`를 노션 DB row로 저장

2. 질문 관리형
- 질문/옵션을 노션 DB에서 관리하고 앱에서 로드

3. 양방향형
- 질문은 노션에서 읽고, 결과는 노션으로 쓰기

## 추후 구현 순서
1. Notion Integration 생성 + DB 생성
2. `.env`에 `NOTION_API_KEY`, `NOTION_DATABASE_ID` 추가
3. `notion_client` 모듈 추가 후 결과 저장 함수 구현
4. 실패 시 로컬 DB fallback 처리
