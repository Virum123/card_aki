# 카드의요정 (Streamlit)

스무고개 문답 방식으로 사용자 소비 성향을 파악해 카드 상품을 추천하는 서비스입니다.

## 서비스 개요

### 1) 타이틀
- 카드의요정

### 2) 기획 의도
- 과거 유행했던 아키네이터와 같은 스무고개 방식의 문답을 통해 사용자에게 가장 적합한 카드상품을 추천하는 시스템.

### 3) 유사 서비스
- 카드고릴라 1분 카드 매칭: https://www.card-gorilla.com/test/intro

### 3-1) 유사 서비스의 한계(정의)
- 발급 유도형 이벤트 중심 구조로 인해 사용자 최적 카드보다 플랫폼 수익성 중심 추천으로 기울 수 있음.
- 카드의요정은 특정 채널 유도보다 사용자 적합성 중심 추천을 우선 목표로 둠.

### 4) 질문 목록
1. 월 사용 금액 or 희망하는 전월실적 수준은 어느정도인가요?
2. 연회비는 어느 정도까지 괜찮으신가요?
3. 해외여행이나 직구를 자주 하시나요?
4. 비행기 마일리지를 모으고 싶으신가요?
5. 베이커리나 카페를 자주 가시나요?
6. 식당에서 자주 결제하시나요? 배달음식을 즐겨 드시나요?
7. 대형마트를 자주 이용하시나요? 온라인 쇼핑을 주로 이용하시나요?
8. 편의점을 주로 가시나요?
9. 통신요금을 직접 납부하시나요?
10. OTT나 스트리밍 서비스를 구독하시나요?
11. 주로 사용하는 간편결제 서비스가 있으신가요?
12. 차를 가지고 계신가요? 혹은 차를 구입하실 생각이신가요?
13. 대중교통을 주로 이용하시나요?

## 데이터/구현 상태
- 수집 데이터: 카드명, 카드사, 요약문구, 국내/해외/가족 연회비, 혜택 태그, 상세 링크
- 추천 방식: 질문 응답 기반 필터링
- 질문 정의 위치: `app/question_bank.py` 의 `QUESTIONS`

## 1) 가상환경 및 의존성
```bash
uv venv .venv
source .venv/bin/activate
UV_CACHE_DIR=.uv-cache uv sync
```

## 2) 크롤링 + DB 생성
```bash
uv run python crawler/build_card_db.py
```

## 3) Streamlit 실행
```bash
uv run streamlit run app/streamlit_app.py
```

기본 링크: http://localhost:8501

## 4) 질문 수정 방법
```bash
# 질문 정의 파일 수정
app/question_bank.py
```

- 질문은 `QUESTIONS`에 추가하면 화면에 자동 노출됩니다.
- 질문 타입: `single`, `multi`, `text`, `range`
